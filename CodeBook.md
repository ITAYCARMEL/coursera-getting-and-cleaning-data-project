# Code Book

Description of `tidy_dataset.txt`, with both train and test dataset

## Factors

* `Subject` - Test subject, represented by a number
* `Activity` - The type of activity performed when the corresponding measurements were taken
** WALKING
** WALKING_UPSTAIRS
** WALKING_DOWNSTAIRS
** SITTING
** STANDING
** LAYING
